Installation

 
In this section, you will learn how to...
- Download the script
- Unzip it
- Drag the correct folder to your desktop
- Upload to your “root” files via FTP
 
Download the script
First, download the BookingWizz application from your CodeCanyon account downloads page and save it to your desktop. This is how the file will look on your desktop:
 
*Note - This is not the file you should upload via your FTP. The file you need is located inside this folder.
 
 
Unzipping the Folder
To unzip the folder and retrieve the file you need, double-click on the CodeCanyon folder. You will see these files:
 

 
 
Next, drag the “booking” folder to your desktop
Here is your “booking” folder on your desktop.
 
 
 
Upload the downloaded script
Now it’s time to upload the “booking” folder to the “root” folder of your website.
 
To get to your FTP files, open your Windows browser and type:
ftp.yourdomainname.com - like so. Be sure to enter your specific domain name.
 
*Note - Some browsers work differently. This document uses the Windows browser. (We
highly recommend using FTP software like FileZilla for example (www.filezilla-project.org))
 
 
 
You will be taken to the following screen, to enter your FTP username and password (if you don't know these - please contact your hosting provider).
 
 
 
When you insert the correct username and password, your browser window will load and show your ftp root files. (See picture below)
 
 
 
Scroll right on the navigation bar and find the word “Page.” Click on it to open a drop-down box
like the one below. At the bottom of the drop-down click  “Open FTP Site".
 
 
You will get this window for a second time:
 
 
 
Insert the correct username and password again, and you will get this “folder” version of your “root” folder:
 
 
 
STOP!
If you are using BookingWizz as a stand-alone program, simply drag the folder into your root.
BUT, if you have shared hosting, the procedure is different. You would drag the booking folder into the domain folder that you will run BookingWizz on. Here’s an example of shared hosting:
 
 
 
There is one main domain name, then other “shared” domains (sometimes called add-on domains in the cpanel). Decide which of these domains you will use to run your BookingWizz application. In this case, we used the folder “Treasures,” which is highlighted in the image above (which is root for domain indieplatinum.com ).
 
To use the folder, we must “double click” it to open to the root files. 
 
Drag the “booking” folder to the desktop
Now we drag the “booking” folder from the desktop to this file, and we have properly installed it into the “root.”
 
 
 
How to check that you've uploaded booking folder to correct place.
 
To be sure you’ve dragged your “booking” folder to the correct path, open a new browser and
type in your domain name and /booking/ . The following image is an example of what you would
see if you have dragged your folder into the correct root file:
 
Notice the alert. It says “Application not installed! Click here to proceed with installation.”
 
 
How to install BookingWizz as a standalone application (non-wordpress)
 
This means that you have dragged your folder to the correct root folder, and now all you need to do is install it as a stand-alone by clicking “Click Here” and following on-screen instructions (Again, it is important to note here, that if you’e going to use BookingWizz for wordpress - you DO NOT need to click on “click Here”and manually install BookingWizz - this will be done automatically for you when you activate BookingWizz for Wordpress plugin)
