New since version 1.2
- events functionality (with paypal integration)
- administrator can setup as many events as he needs for any day. You can use this functionality for events, meetings, seminars, hotel rooms, etc. When creating event you have option to set spaces for event, cost for entry and required or not required payment.
- when somebody reserves a spot in event and pays for the entry – admin will receive notice.
- paypal currency and paypal email can be changed in settings.
- if you want to accept credit cards on paypal payment page – you need premium or business account from paypal (free)
- we have also added captcha’s to all forms (no-js and js). Cancellation email was also added for customers.
